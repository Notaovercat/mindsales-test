# Mindsales-test
Тестовое задание на позицию Fullstack Node.js/Vue Developer в ТОО MINDSALES

## Подготовка
### Требования
```bash
npm i -g pnpm
```
### Установка зависимостей
```bash
pnpm i
```
## Запуск
### Запуск в dev режиме
```bash
pnpm run back:build
pnpm run dev
```
### Запуск в prod режиме
```bash
pnpm run build

# Запуск сервера
pnpm run back:start

# Для запуска клиента нужен http-server или аналог
# npm i -g serve
serve frontend/dist
```
